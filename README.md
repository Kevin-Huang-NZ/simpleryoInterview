# simpleryoInterview

# 一、环境
1、目录说明

  该程序实现分为前、后端两个工程。
  shop: 是后台java工程。
  shop-frontend：是前端reactjs工程。
  
2、开发运行环境

  OS: windows 7（其他操作系统和版本未测试）
  java version "1.8.0_131"
  Apache Maven 3.5.2
  nodejs v8.9.1
  npm 5.5.1
  yarn 0.20.4
  
# 二、运行
1、java工程
  在shop工程根目录下，命令行执行：mvnw spring-boot:run

2、前端工程
  在shop-frontend工程根目录下，先执行yarn安装依赖包。
  再执行npm run dev

3、访问
  http://localhost

4、注意
  该工程运行需要占用8080和80两个端口，需保证不冲突。
